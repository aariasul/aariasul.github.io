INTER-ACTIVEID REUSABLE TEMPLATE

Files:
1. MASTER-INTER-ACTIVEID-SOP.md
   Detailed instructions for an AI or developer.

2. business-config.template.json
   Fill this with the data for each new business.

Recommended workflow:
- Copy the JSON template.
- Rename it for the new business.
- Fill every available field.
- Give the SOP, completed JSON, latest Inter-ActiveID files, visual reference files, and target assets to the AI.




Para un nuevo negocio:

1. Copia business-config.template.json.
2. Renómbralo, por ejemplo:
  business-config-acme.json
3. Completa sus datos.
4. Entrega a la IA:
  MASTER-INTER-ACTIVEID-SOP.md
  El JSON completado.
  El ZIP más reciente de Inter-ActiveID.
  El ZIP de la tarjeta usada como referencia visual.
  Logo, fotografía, vCard e imágenes del nuevo negocio.

Con eso, la IA tendrá instrucciones suficientemente precisas para repetir el mismo proceso que hicimos con Materia y Lumi.
